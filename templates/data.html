Stroke Dataset

Informasi data

Data yang kami digunakan yakni Stroke Dataset kami peroleh dari tersebut dari Kaggle. Yang didalamnya memuat sebagai berikut:

  • Gender = jenis kelamin pasien
  • Age = Usia pasien (tahun)
  • Hypertension = Hipertensi (1 = hipertensi, 0 = tidak)
  • Heart disease = Riwayat sakit jantung pasien (1 = sakit jantung, 0 = tidak)
  • Ever married = Status pernikahan pasien
  • Work type = Jenis pekerjaan pasien
  • Residence type = Jenis tempat tinggal pasien
  • Avg glucose level = Kadar gula darah pasien
  • BMI = Indeks massa tubuh pasien
  • Smoke status = Riwayat merokok pasien
  • Stroke = Riwayat stroke pasien ( 1 = stroke, 0 = tidak)
